# GIS (地理情報システム)

## データ形式
- ベクターデータ
    - GeoJson
    - shpファイル
- ラスターデータ
    - GeoTIFF

## 地図タイル
タイルの概念、特にベクタータイルの登場によって、配信のコストが大幅に下がった (mapbox, maplibreのようなwebGLベースの描画とあわせて、高度なクライアントサイドレンダリングが可能となった。)

ラスター・ベクターの両形式で、タイルのデータが存在する。
ラスタータイルは、QJISやgdalで変換できる。
ベクタータイルは、`tippecanoe` を使うことで、`mvt`形式に変換できる。


## 表示の基本
- Layerを重ねて、情報を表現する
